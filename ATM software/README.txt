Основной функционал:
1. Верификация. Проверка на наличие номера карты и последующей проверки пин-кода
2. Проверка баланса по валюте.
3. Внесение средств.
4. Выдача средств.

Сообщения, которые выводятся хранятся в файлах .properties
Номера карт, пин-коды хранятся в файле .properties(в планах подключить БД)
